Prey-Ball


How to play ?

1. User can use keyboard to control the direstion.
2. User can change the difficulty by selecting speed and time.
3. Red ball is bomb, if user eat the bomb, the ball will get small
4. Click the start button to start a game and start timing
5. Click the reset button to resize the ball and reset the timer
6. If the ball get big enough, the page will alert a winner message
7. If the time is up, the page will alert a box and asking restart
